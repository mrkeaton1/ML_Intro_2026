## First Steps: Installing software

This workshop is designed to provide an introduction to the world of machine learning. To get started, it's important to be familiar with some of the common tools used for developing machine learning models, and programming in general. Since this workshop doesn't assume any prior experience, let's quickly step through the important programs you will need.

1. An **IDE**, or interactive development environment. There are many common tools: VSCode, PyCharm, Cursor, IDLE, Spyder, Neovim, etc. In case you don't have one installed yet, I recommend VSCode as a solid starting place, as it has the largest community, tons of plugins, and is fairly beginner-friendly. https://code.visualstudio.com/download
2. **git**. If you want to share code or use someone else's, you will undoubtedly utilize git at some point. git is a very powerful open-source **version control** system that may be a bit daunting at first, but we can get away with a few commands to do the majority of what we would want for our first codebase. https://git-scm.com/install/
3. **Python**. Many computers come with Python pre-installed, but in case you don't have it on yours yet, it's easy enough to download from the official website. https://www.python.org/downloads/
4. Python and jupyter **extensions**, if using VSCode. These will likely be prompted to installed if you are coming from a fresh install of VSCode, but if you have any issues, I can walk you through it.
